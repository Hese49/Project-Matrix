Project_matrix

An educational module for understanding how partial differential equations (PDEs) can be solved using linear algebra methods.

Overview
This project is designed for engineering students with background in:
- Calculus
- Linear algebra
- Differential equations
- Basic Python

The goal is to build a clear conceptual bridge:
PDE > Discretization  > Matrix formulation  >  Linear algebra insight

 Contents
- Heat Equation (Parts A...D)
- Wave Equation
- Poisson Equation
- 2D Poisson Equation
- Iterative Solvers
- Sparse Matrices
- Schrödinger Equation

 Key Ideas
- PDEs can be transformed into algebraic systems
- Matrices encode physical structure
- Numerical methods approximate solutions
- Linear algebra reveals deeper insights

Usage
Each notebook is self-contained but designed to be followed in order.

Philosophy
This project emphasizes:
- intuition over formalism
- structure over computation
- clarity over complexity
